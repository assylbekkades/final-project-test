# Final Project - Test Assignment

## Описание
Ответы на вопросы тестового задания по аналитике данных.

## Содержание
- Теоретические вопросы по SQL
- Теоретические вопросы по Python
- Теоретические вопросы по статистике
- Теоретические вопросы по машинному обучению
- Практические задачи с решением

## Инструменты
- Python 3.x / Jupyter Notebook
- MySQL / MySQL Workbench
- Microsoft Excel

## Структура файлов
- `test_answers.docx` — Word документ с ответами на вопросы
- `test_solution.ipynb` — Jupyter Notebook с решением задач
- `test_solution.sql` — SQL файл с решением задач

## Навыки
- SQL: JOIN, GROUP BY, Window Functions, CTE
- Python: pandas, numpy, matplotlib, scikit-learn
- Статистика: гипотезы, распределения, доверительные интервалы
- ML: регрессия, классификация, метрики качества
